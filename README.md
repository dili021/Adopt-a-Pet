## About

A SPA built from the ground up with React, which is used to search for pets of all kinds that are available for adoption.

## Built with

- React
- Reach router
- Parcel
- ESlint
- Babel

## Setup

- Clone the repo to your computer
- Run `npm install`
- Run `npm run dev`
- Visit `localhost:1234` in your browser
